# CLI Banking App (Java)

A command-line based banking application built using **Java**, **JDBC**, and **SQLite**.

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Check account balance
- Persistent storage using SQLite

## Technologies Used
- Java
- JDBC
- SQLite
- Command Line Interface (CLI)

## How to Run
1. Clone the repository
2. Add SQLite JDBC to classpath
3. Compile and run `BankingApp.java`

## Purpose
This project was built to strengthen core Java, database connectivity, and backend logic fundamentals.
